# Automatisation PowerShell & Microsoft Graph

## Contexte

Certaines tâches d'administration récurrentes nécessitaient des actions manuelles régulières.

L'objectif était de les automatiser.

---

## Technologies utilisées

- PowerShell
- Microsoft Graph
- Microsoft Intune
- Windows Autopilot

---

## Cas d'usage

### Gestion Autopilot

Mise à jour en masse des Group Tags Windows Autopilot.

### Synchronisation horaire

Remédiation automatique du service de temps Windows sur les postes administrés.

### Configuration système

Modification automatisée de paramètres système et de clés de registre.

---

## Résultats

- Réduction des interventions manuelles
- Standardisation des configurations
- Simplification de l'exploitation

---

## Compétences démontrées

- PowerShell
- Microsoft Graph
- Automatisation
- Intune
